# クリーンDB初期化→migrate→seed プロンプト

このプロンプトは、データベースを初期化し、マイグレーションを実行し、シードデータを投入する手順を定義します。

## 手順

1. **データベースの初期化**
   - データベースをクリーンな状態にリセットします。
   - 既存のデータやスキーマを削除し、新しいデータベースを作成します。

2. **マイグレーションの実行**
   - 最新のマイグレーションファイルを適用し、データベーススキーマを更新します。
   - 必要なテーブルやインデックスを作成します。

3. **シードデータの投入**
   - 初期データをデータベースに投入します。
   - これにより、アプリケーションが動作するために必要な基本的なデータが設定されます。

## コマンド例

以下は、上記の手順を実行するためのコマンドの例です。

```bash
# データベースの初期化
npm run db:clean

# マイグレーションの実行
npm run db:migrate

# シードデータの投入
npm run db:seed
```

このプロンプトを使用することで、データベースのセットアップが簡単に行えます。
