# 日鋼団地の計画

住戸希望を集めて、団地全体の人気や配置の偏りを見やすくする Web アプリです。
参加者は住戸を複数選んで希望を出せます。全体画面では人気の集中を確認でき、
必要に応じて部屋ごとの調整チャットも使えます。

## インストール

このリポジトリは `apps/web` 配下がアプリ本体です。

```bash
cd apps/web
pnpm install
```

## 起動

```bash
pnpm dev
```

ブラウザで `http://localhost:3000` を開きます。

## よく使うコマンド

```bash
pnpm build
pnpm start
pnpm lint
```
