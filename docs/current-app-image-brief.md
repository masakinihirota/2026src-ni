# 日鋼団地 住戸希望チェックのイメージ図向け説明

## この文書の用途

この文書は、利用者向けの紹介図を作るための文章です。  
設計情報ではなく、見た人が使ってみたくなるイメージを伝えることを目的にしています。  

## 伝えたい体験

このアプリは、住戸選びの不安を減らすための画面です。  
利用者は住戸番号を入力して参加します。  
ただし他の人に見える表示は星座匿名名です。  
たとえば「9-105」の利用者が、画面上では「青い穏やかな水瓶座」と表示されます。  
番号は管理しつつ、公開では匿名にするので安心して参加できます。  

## 画面の雰囲気

全体は白と薄いグレーを基調にした、落ち着いたUIです。  
要所に青系アクセントを使い、操作ボタンや見出しを分かりやすくしています。  
派手さより、見やすさと迷いにくさを優先した印象です。  

## 図で見せたい構図

1枚のイメージ図で次の要素を同時に見せます。  

- 左側に住戸図面のグリッド。  
- 住戸セルに人気件数バッジ。  
- 下側に第1から第3希望のカード。  
- 右側に選択中住戸の詳細パネル。  
- 上部か注釈に「9-105 → 青い穏やかな水瓶座」の匿名化ラベル。  
- 別領域に全体ヒートマップの小さなサムネイル。  
- 匿名表示のチャット導線。  

## 見る人に伝えたいメッセージ

住戸を直感的に選べること。  
人気の重なりを一目で確認できること。  
匿名表示で安心して調整に参加できること。  
この3点が伝われば、このアプリの魅力が伝わります。  

## 画像生成AI向け短文

日本の団地住戸選定Webアプリ紹介イメージ。  
白と薄いグレー基調、青系アクセント、落ち着いて見やすいUI。  
住戸番号入力後に公開表示が星座匿名名へ変わる流れを表現。  
例として「9-105 → 青い穏やかな水瓶座」を注釈表示。  
左に住戸図面グリッドと人気バッジ、右に住戸詳細、下に第1〜第3希望カード。

スマホとPCの両方を想定した、安心感のある構図。
